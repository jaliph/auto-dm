QR Code Display Example

How to Display QR Code PNG

When you get a QR code response from the API, you can display the PNG image like this:

<img src="data:image/png;base64,{qr_code_png}" alt="QR Code">

Replace {qr_code_png} with the actual base64 string from the API response.

API Usage

To get a QR code:

  1. Register a sender: POST /register
  2. Get QR code: GET /qr/{token}
  3. Display the PNG using the base64 data
